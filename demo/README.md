# where-log demo

## 1) Install demo deps

```bash
cd demo
npm install
```

## 2) Run CommonJS demo

```bash
npm run cjs
```

## 3) Run ESM demo

```bash
npm run esm
```

## 4) Run both

```bash
npm run all
```

## 5) Preset demo

```bash
npm run presets
```

## 6) Level demo

```bash
npm run levels
```

## 7) Once + Time + Context demo

```bash
npm run session
```
